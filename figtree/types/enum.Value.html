A type to represent a figtree value

Generally, this is obtained by getting the value of an attribute on a Node, although there are also methods to construct all the kinds of this type.

Examples

let value = Value::new_string("hello!");
assert!(value.get_str() == Some("hello!"));
assert!(value.get_int() == None);
